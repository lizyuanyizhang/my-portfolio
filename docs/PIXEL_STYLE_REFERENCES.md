## 四、设计要点（借鉴 8-bit 游戏质感）

### 视觉
- **像素块**: 边缘用方形像素块，避免抗锯齿
- **配色**: 亮红、明黄、深蓝等高饱和，贴近 FC/NES
- **字体**: 像素字体，保留明显像素边缘
- **背景**: 格子、砖块、云朵等像素化场景元素

### 交互
- **Hover**: 像素位移 + 颜色变化，模拟实体按键
- **点击**: 高亮待命效果，可配合音效
- **动画**: 帧动画、逐帧移动，而非平滑过渡

### 品牌融合
- 像素元素与整体调性统一，避免孤立
- 可与复古元素（老照片、打字机、黑胶）混搭
- 保持呼吸感，不要过度堆砌

## 六、与现有首页结合的思路

1. **保留 Brutalist 粗边框** → 可改为像素化边框（阶梯式而非圆角）
2. **字体** → 标题用 Press Start 2P 或 VT323，正文可保留 Serif 做对比
3. **配色** → 引入 8-bit 调色板（如深绿、墨绿 + 米白）
4. **背景** → 淡色格子或 subtle 像素纹理
5. **按钮 / 链接** → 像素风 hover（位移、变色）
6. **动画** → 生命计时器、访客数等可做像素化数字切换

如需下一步，可从「改字体 + 配色」最小改动开始，再逐步加入像素化组件。
